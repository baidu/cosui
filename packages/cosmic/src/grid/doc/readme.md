# Grid 栅格

- 栅格系统：是一种使用行和列来定义布局的方式。
    - 使用 Flex 布局
    - 支持响应式布局
    - 支持自定义总栅格列数
- 覆盖平台：PC/Mobile

## 响应式布局

预设六个响应尺寸：`xs`/`sm`/`md`/`lg`/`xl`/`xxl`。

|响应尺寸|xs|sm|md|lg|xl|xxl|
|---|---|---|---|---|---|---|
|默认尺寸范围|<500px|<660px|<990px|<1200px|<1500px|>=1500px|
|建议栅格列数|12列|12列|24列|24列|24列|24列|

- xs 对应搜索 Mobile 下的栅格
- sm 对应搜索 PC 下的栅格
