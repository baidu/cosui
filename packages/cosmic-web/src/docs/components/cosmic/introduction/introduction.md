# 概述
Cosui 是一套基于 San 框架 的、面向 大模型与智能体应用 的 UI 组件库。
通过支持 Markdown、JSON、San 模板等多协议驱动 UI 渲染，适用于 生成式 UI、智能体界面、AI 对话系统 等新型人机交互场景。

## 核心优势
### 面向 AI 与 LUI 的 UI 能力
* **多协议驱动 UI**：支持 Markdown、JSON、San 模板等多种协议接入，可由大模型或服务端动态生成 UI 描述
* **LUI 场景友好**：支持与服务端共建数据结构，灵活承载生成式 UI、对话式交互、工具面板等复杂场景

### 开箱即用的工程能力
* **组件丰富**：内置 80+ 基础 UI 与复合业务组件，覆盖常见大模型应用与中后台场景
* **现代构建体系**：提供 ESM 构建产物，支持 Tree Shaking 与按需加载，减少最终包体积
* **完善的类型校验**：采用 TypeScript 开发，提供完整类型定义，提升可维护性与稳定性
* **高度可定制**：原子组件提供多种变体、插槽与配置能力，满足复杂业务需求
* **样式工程化**：所有样式均以组件为命名空间，避免全局样式污染，提升可维护性与可读性

### 主题与多端适配
* **多主题切换**：基于 CSS Token 的主题体系，内置亮色 / 暗黑模式，支持业务自定义主题
* **双端适配**：同时支持 PC 与移动端，适配不同终端与交互方式
