## 概述

**Cosmic Dynamic UI** 是上述 [JSON 协议](/protocol/json/content/index) 的标准参考实现。它是一个面向 AI 生成场景的 **JSON-to-UI 渲染引擎**。

## 核心特性
*   **完备的协议解析**：完整支持协议元语，内置 `if`/`for` 逻辑控制及复杂的插值表达式引擎，实现灵活的数据绑定与运算。
*   **AI 流式数据驱动**：支持 SSE 增量数据追加 (`appendData`)，适应 AI 输出特性。
*   **高可扩展组件体系**：采用“自身无 UI”设计，通过映射机制将协议节点转化为具体业务组件，实现协议层与渲染层的解耦。
*   **交互与事件闭环**：构建完整的交互链路，支持自定义 **Action Handler** 注册与上下文参数自动解析，灵活处理业务逻辑。

## 功能说明
面向开发者提供 UI 协议解析能力

* 渲染：渲染能力
* 编译&解析：
  * 协议基础解析能力
  * 解析 UI 能力
  * 编译 dataExtends 能力
* 数据：二次追加数据，数据分段渲染能力
* 组件：自定义组件，对应扩充 UI 协议的 节点类型 能力
* 行为：自定义行为，对应扩充 UI 协议的 action 能力
* 事件：自定义事件，对应处理组件自定义事件
